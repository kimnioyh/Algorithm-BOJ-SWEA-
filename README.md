2. ### 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

    https://www.acmicpc.net/problem/1000


    C++
    ```C++ 
        #include <stdio.h>

        int main()
        {
            int i, j;
            scanf("%d %d",&i,&j);
            printf("%d",i+j);
        }
    ```

    Python
    ```python
    num1, num2 = map(int, input(). split())
    print(num1 + num2)
    ```
